# Storefront Template

(opens new window) (opens new window) (opens new window)

PWA & JAMstack e-commerce template for E-Com Plus (not only) stores, mainly composed by:

  1. @ecomplus/storefront-framework (opens new window) Webpack based compiler;
  2. @ecomplus/storefront-twbs (opens new window) base UI system;
  3. @ecomplus/storefront-components (opens new window) Vue components library;
  4. @ecomplus/storefront-app (opens new window) Vue SPA for cart, checkout and account page;
  5. Other E-Com Plus related libraries and additional widgets;
